== Installation ==

Installing this plugin as regular wordpress plugin. First you need to install Visual Composer plugin then install Maria Hover Effects - Visual Composer Extension.

This addon plugin works like other page builder addon plugin. Go to new page then click on Visual Composer builder then click on Add Element you should see Maria Hover Effects menu as Name "Maria Hover Effects" on Menu Top, click on it you will see Maria Hover Effects then click on it and browse an image and customize according to your needs.
